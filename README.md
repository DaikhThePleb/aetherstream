# AetherStream

AetherStream is a desktop app for streamers who want clear, natural text-to-speech from Twitch chat.

It helps you read chat messages aloud, apply voice rules for rewards and users, and keep your stream overlay in sync.

## What You Can Do

- Read Twitch chat messages with Azure voices.
- Set global voice style, speed, pitch, and volume.
- Create custom voice rules for Channel Point rewards.
- Create custom voice rules for specific users.
- Filter spam, links, blocked words, and blocked users.
- Show live status in OBS through a local overlay URL.
- Save and import/export presets.

## Quick Start

1. Download the latest installer from the Releases page.
2. Install and open AetherStream.
3. Enter your Azure Speech key and region.
4. Connect your Twitch account.
5. Pick your default voice and test playback.
6. (Optional) Add the local overlay URL to OBS.

## Updates

- The app can check for new releases directly from GitHub.
- If an update is available, you can review the changelog and start the installer from inside the app.

## Privacy and Safety

- Your local settings are stored on your machine.
- Sensitive values should be kept in local environment files and never committed.
- The updater only accepts trusted HTTPS GitHub hosts for installer downloads.
